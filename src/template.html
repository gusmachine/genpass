Usage

  1. Click "Show Salt" and input random characters (preferrably > 50 characters) in the box. Then create a bookmark of this.
  2. Decide and type your "master" password. It will be the password "seed" to generate a lot of password for you.
  3. Type the hostname for which you want to use the generated password, and choose the character set you want to use from the "Char" list.
  4. Click "Generate". A password is generated at the bottom boxes in invisible color. Copy it. Or click "Show" to see it.
  5. The password will disappear when the counter reaches 0. You can just "generate" again to see it again.
