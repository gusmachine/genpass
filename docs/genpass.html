Usage

  1. Click "Show Salt" and input random characters (preferrably > 50 characters) in the box.
  2. Type your password.
  3. Type the hostname for which you want to use the generated password, and choose the character set you want to use from the "Char" list.
  4. Click "Generate". A password is generated at the bottom boxes in invisible color. Copy it. Or click "Show the output" to see it.

Tips

  • Bookmark this page after typing the salt string. The salt will be filled in the bookmark.
  • The text next to the password is generated from your password and the salt. Please check it to see your password is correct.
  • The password will disappear after 20 sec, when the counter reaches 0. You can just "generate" again to show it again.
